# DJangoReactNoteApp
DJangoReactNoteApp

Welcome to the DJangoReactNoteApp!

A simple note web-app that uses Django as the backend which gives React, the frontend access to its database to create, update, and delete notes through a RESTful API.

Feartures
* adding
* creating 
* editing  
* deleting
